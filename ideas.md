# 株式会社堺設備 ウェブサイト デザイン構想

## 選定デザイン: モダン・プロフェッショナル（青系グラデーション）

### デザイン哲学
参照サイトから抽出した**青系グラデーション + 幾何学的な斜めカット**という強い視覚的アイデンティティを忠実に再現します。水道設備という実用的な産業を扱いながら、信頼感と革新性を同時に表現する洗練されたデザイン。

### コアプリンシプル
1. **グラデーション主導**: 明るい水色（#4DB8E8）から深い青紫（#5B5FDE）へのダイナミックなグラデーション
2. **幾何学的構成**: 斜めカット（clip-path）で空間を分割し、モダンで動的な印象を創出
3. **信頼と実用性**: 清潔感のある白背景とのコントラストで、プロフェッショナルさを強調

### カラーパレット
- **プライマリ青**: #4DB8E8（明るい水色）
- **セカンダリ紫**: #5B5FDE（深い紫青）
- **背景**: #FFFFFF（白）
- **テキスト**: #333333（濃いグレー）
- **アクセント**: グラデーション領域

**理由**: 水道設備の「清潔さ」と「信頼性」を青系で表現。グラデーションは企業の「成長」と「革新」を象徴。

### レイアウトパラダイム
- **セクションごとの斜めカット分割**: 各セクション（ヒーロー、インタビュー、事業内容など）を斜めのクリップパスで区切る
- **非対称配置**: テキストと画像を交互に配置し、視線の流れを作る
- **ホワイトスペース活用**: グラデーション領域と白背景を交互に配置

### シグネチャー要素
1. **斜めカット背景**: 各セクションの背景に45度の斜めカットを適用
2. **グラデーション背景**: 青系グラデーションをセクション背景に使用
3. **ジオメトリック図形**: 三角形やポリゴン形状でセクション区切りを表現

### インタラクション哲学
- **スムーズなスクロール**: セクション間の移動を自然に
- **ホバーエフェクト**: ボタンやリンクにスケール・色変化を適用
- **段階的な表示**: スクロール時に要素がフェードイン

### アニメーション
- **セクション遷移**: 斜めカットの境界でスムーズなトランジション（300ms）
- **ボタンホバー**: スケール（0.98→1.02）+ 色変化（200ms ease-out）
- **テキスト表示**: スクロール時にフェードイン + スライドアップ（400ms）
- **グラデーション**: 背景グラデーションは静止（動的グラデーションは避ける）

### タイポグラフィシステム
- **ヘッドライン**: 日本語フォント（Noto Sans JP Bold）+ 大きめサイズ（32px-48px）
- **サブヘッドライン**: Noto Sans JP Medium（20px-24px）
- **ボディテキスト**: Noto Sans JP Regular（14px-16px）
- **キャッチコピー**: Noto Sans JP Light（18px-24px）で軽やかに

**理由**: 日本語の可読性を保ちながら、モダンな印象を与える。

## 実装ガイドライン

### CSS変数
```css
--color-primary: #4DB8E8;
--color-secondary: #5B5FDE;
--gradient-primary: linear-gradient(135deg, #4DB8E8 0%, #5B5FDE 100%);
--clip-angle: polygon(0 0, 100% 8%, 100% 100%, 0 92%);
```

### コンポーネント設計
- **HeroSection**: グラデーション背景 + 斜めカット + テキストオーバーレイ
- **InterviewCard**: 画像 + テキスト + ホバーエフェクト
- **SectionDivider**: 斜めカット形状で次セクションへの視覚的つながり
- **CTAButton**: グラデーション背景 + ホバーエフェクト

### レスポンシブ設計
- **モバイル**: 斜めカット角度を緩くする（8% → 5%）、フォントサイズを調整
- **タブレット**: 標準設定
- **デスクトップ**: 標準設定 + より大きなグラデーション領域

## 参照サイト分析

### 構成要素
1. **ヘッダー**: ロゴ + ハンバーガーメニュー（モバイル）
2. **ヒーロー**: 建物写真 + キャッチコピー + グラデーション背景
3. **インタビューセクション**: 社員写真 + 職種情報 + カード形式
4. **事業内容**: テキスト + 工具画像 + グラデーション背景
5. **採用情報**: 職種カード + 詳細リンク
6. **フッター**: 会社情報 + リンク

### 色使い
- 背景: 白 + グラデーション青紫
- テキスト: 濃いグレー + 白（グラデーション領域内）
- ボタン: グラデーション青紫 + 白テキスト

### 画像配置
- ヒーロー: 建物全体写真（大きく表示）
- インタビュー: 社員顔写真（正方形カード）
- 事業内容: 工具・設備写真（右側配置）
- 採用: 作業車写真（左側配置）
